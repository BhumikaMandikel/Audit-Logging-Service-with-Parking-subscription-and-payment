About This Dashboard

This dashboard provides a user-friendly interface to interact with the PES University Microservices system.

Features:

  • Test subscription service endpoints
  • View and filter audit logs
  • Real-time response visualization
  • User-friendly interface

System Components:

  • Subscription Service: Handles subscription creation and payment processing
  • Log Microservice: Centralized logging for all system activities
